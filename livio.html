📧 Email Setup — Direct Send via EmailJS
How to enable direct email sending:
1. Go to emailjs.com and create a free account (200 emails/month free)
2. Add an Email Service (Gmail, Outlook, etc.) → copy your Service ID
3. Create an Email Template using these variables: {{to_email}} {{subject}} {{message}} → copy your Template ID
4. Go to Account → API Keys → copy your Public Key
5. Paste all three below and click Save
Found in EmailJS → Email Services
Found in EmailJS → Email Templates
Found in EmailJS → Account → API Keys

📋 EmailJS Template Setup Guide

When creating your EmailJS template, use these exact variable names so the software fills them automatically:

Template Variable What it contains
{{to_email}}Recipient email address
{{to_name}}Recipient name / vendor name
{{subject}}Email subject line
{{message}}Full email body / ledger text
{{from_name}}Sender name (Livio Building Systems)
{{reply_to}}Your reply-to address
⚠ Tip: In your EmailJS template, set To Email field to {{to_email}} and the body to {{message}}. EmailJS free plan allows 200 emails/month.
